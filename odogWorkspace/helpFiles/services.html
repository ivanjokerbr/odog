Services


The following services are available for:

All components


int numberOfConnections(char *port)

    Returns the number of connections associated with port.


char *nameOfConnection(char *port, int number)

    Returns the name of the connection of port port whose number is specified by the parameter number.



ISEM associated with value DE

void scheculeMe(double instant) 

    Requests the execution of the atomic component at instant. If multiple calls to this service are made prior to reaching the least value of instant specified by them, only the last one will be efective.


void abortSchedule(void) 

    Invalidates the previous call to the scheduleMe().


double currentTime(void) 

    Returns the current value of time.




ISEM associated with value SR

void setAbsent(char *con)
   
    Sets the state of the connection specified by con as being known and absent in the particular instant.


void setAbsentAll(char *port) 

    Sets the state of all connections associated with the input port port as being known and absent in the particular instant.


char isAbsent(char *con) 

    Returns 0 if the state of connection con is known and absent. Otherwise returns 1.


char isAbsentAll(char *port) 

    Returns 0 if the state of all connections associated with input port port is known and absent. Otherwise returns 1.

The canReceive and canReceiveAll methods will return 1 only when the state of an input connection is different than unknown. The canSend and canSendAll methods will return 1 only when the state of an output connection is unknown.



ISEM associated with value DF
void setSampleRate(char *port, short value)

    Sets the sample rate of port.


int getSampleRate(char *port)

    Returns the sample rate of port.


void stopExecution(void) 

    Asks for immediate termination of the execution. The finish methods of all components will be called before exiting.
